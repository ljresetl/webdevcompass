## 🌟 Про проект
Це моє персональне портфоліо — сучасний "роздоріжник" (hub) для моїх проектів. Головна мета — продемонструвати поєднання естетичного дизайну та бездоганної технічної реалізації.

### Що робить цей сайт особливим:
* **⚡️ Екстремальна швидкість:** Показник **95+ балів** у Google Lighthouse.
* **🛠 Чиста архітектура:** Використання TypeScript та модульного SCSS для масштабованості.
* **🔍 SEO-First:** Сайт повністю оптимізований для пошукових систем та соціальних мереж.
* **📱 Адаптивність:** Ідеальний досвід користувача на будь-якому пристрої від 320px до 4K.
